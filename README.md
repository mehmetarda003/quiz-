# Quiz Arena

Kahoot tarzinda canli bilgi yarismasi web sitesi.

## Ozellikler
- Sinirsiz soru ekleme (host panelinden dinamik)
- Canli katilim (Socket.IO)
- 6 haneli sayisal oda kodu
- Odaya ozel QR kod
- Coktan secmeli soru sistemi
- Mobil uyumlu arayuz
- Telifsiz, SVG tabanli avatar tasarimi

## Kurulum
```bash
python -m pip install -r requirements.txt
python app.py
```

Ardindan tarayicida `http://127.0.0.1:5000` adresini acin.
